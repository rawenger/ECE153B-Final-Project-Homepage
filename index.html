Overview

This project aims to implement a bot for the Cup Pong game in the iOS GamePigeon app. It will use the nucleoâ€™s USB peripheral mode to act as a mouse connected to an iPad over USB type-C, and I2C to connect to a Wii nunchuk which would allow a user to move the cursor to select a cup to target. There will also be an automatic mode, where it just fires at all the default targets, but the manual control allows the bot to be used when the cups are placed randomly. If time permits, I will also implement a settings/instructions GUI on an LCD screen.
